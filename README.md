# 🏦 Simple Banking System in Python

A simple command-line Banking System project built with Python.
This project allows users to deposit money, withdraw money, and check account balance.

---

## 📌 Features

* Deposit Money
* Withdraw Money
* Check Current Balance
* Menu Driven Program
* Simple and Beginner Friendly

---

## 🛠️ Technologies Used

* Python 3

---

## 🚀 How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Python-Banking-System.git
```

### Step 2: Open the Project Folder

```bash
cd Python-Banking-System
```

### Step 3: Run the Python File

```bash
python banking_system.py
```
